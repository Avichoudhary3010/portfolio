## Sections

1. **Hero Section** - Introduction with call-to-action buttons
2. **About** - Personal information and statistics
3. **Skills** - Technical skills with progress bars
4. **Projects** - Featured projects showcase
5. **Contact** - Contact form and information

## Customization

### Personal Information
Edit the following in `index.html`:
- Replace "Your Name" with your actual name
- Update the subtitle and description
- Add your social media links
- Update contact information

### Skills
Modify the skills section in `index.html` to match your expertise:
- Update skill names
- Adjust progress bar percentages (data-width attribute)

### Projects
Replace the example projects with your own:
- Update project titles and descriptions
- Add your project links
- Modify technology tags

### Colors
Customize the color scheme in `styles.css` by modifying the CSS variables in the `:root` selector:
- `--primary-color`: Main brand color
- `--secondary-color`: Secondary brand color

## Usage

1. Open `index.html` in a web browser
2. Customize the content to match your information
3. Deploy to your preferred hosting service
